Available API Requests

Locations

  • curl -X GET http://localhost/locations/

  • curl -X POST http://localhost/locations/ -H "Content-Type: application/json" -d { "location_name": "UBC", "coordinate_latitude":5, "coordinate_longitude":56, "fun_facts":"TEST1", "related_links":"TEST1", "about":"TEST1", "image_url":"TEST1" }

  • curl -X GET http://localhost/locations/location_name

  • curl -X PUT http://localhost/locations/location_name -H "Content-Type: application/json" -d { "location_name": "UBC", "coordinate_latitude":5, "coordinate_longitude":56, "fun_facts":"TEST1", "related_links":"TEST1", "about":"TEST1", "image_url":"TEST1" }

  • curl -X DELETE http://localhost/locations/location_name

  • curl -X GET "/locations/user/:user_account_id"

Messages

  • curl -X GET http://localhost/messages/

  • curl -X POST http://localhost/messages/ -H "Content-Type: application/json" -d { "coordinate_latitude":5.35, "coordinate_longitude":14, "message_text":"HELLO", "user_account_id":10 }

  • curl -X GET http://localhost/messages/id

  • curl -X PUT http://localhost/messages/id -H "Content-Type: application/json" -d { "coordinate_latitude":5.35, "coordinate_longitude":14, "message_text":"HELLO", "user_account_id":10 }

  • curl -X DELETE http://localhost/messages/id
